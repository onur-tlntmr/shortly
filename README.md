# Shortly
Bağlantı kısaltma uygulamasını simüle etmektedir. Arayüz ve mantıksal olarak tamamlanmış olup herhangi bir Api ve veritabanı bağlantısı bulunmamaktadır.




# İndirme

```

git clone https://github.com/onurcantimur/shortly.git

```


# Kullanılan Teknolojiler ve Teknik Seçimler
Uygulama flutter ile geliştirilmiş olup, arayüzler arasında veri paylaşımı, Provider paketi kullanılarak yapılmıştır (daha fazlası için bknz. [flutter için provider](https://pub.dev/packages/provider) ).
